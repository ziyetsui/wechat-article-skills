## 文章 Skill

### 风格指纹

- 内容类型：观点型解释分析
- 主题域：商业/创业/增长、投资/金融/加密、社会/历史/文化
- 开头机制：用数字、收益或结果开场，把可信度和阅读收益前置。
- 结构骨架：多章节长文，约 12 个小节，适合用罗马数字/编号标题推进。
- 段落节奏：中短段为主，解释和行动句交替，适合公众号快读。
- 语气判断：解释型：克制但有判断，先拆问题，再给框架和可执行建议。
- 小标题样本：这不是什么新套路。科技行业只是最后才跟上的。 / 数据逻辑 / 瓶颈在于源内容 / 第一步：建立内容工厂 / 第二步：为剪辑而设计内容 / 第三步：在分发之前进行筛选
- 文本规模：约 2889 字符，52 个段落，12 个标题，3 个列表项，0 个引用块
- 爆款承诺：科技巨头正在摒弃传统广告，转而采用病毒式 “切片” 策略——以下是主导社交媒体信息流的秘诀。

```text

写作方式：
1. 开头：用数字、收益或结果开场，把可信度和阅读收益前置。
2. 类型：采用「观点型解释分析」；主题域优先靠近「商业/创业/增长、投资/金融/加密、社会/历史/文化」。
3. 结构：多章节长文，约 12 个小节，适合用罗马数字/编号标题推进。
4. 节奏：中短段为主，解释和行动句交替，适合公众号快读。
5. 语气：解释型：克制但有判断，先拆问题，再给框架和可执行建议。

8. 每个核心段落都要回答一个隐含问题：读者为什么现在要关心、怎么判断、下一步怎么做。

避免：
- 不要写成泛泛鸡汤或百科说明。
- 不要把标题里的承诺摊薄，正文必须持续兑现「科技巨头正在摒弃传统广告，转而采用病毒式 “切片” 策略——以下是主导社交媒体信息流的秘诀。」。
- 不要堆砌过多并列概念；每一节都要服务于主线。
- 不要使用夸张营销话术替代具体机制。
```

## 封面 Skill

### 风格指纹

- 画面类型：细节密集的拼贴/界面/图表视觉
- 原图尺寸：1280x512，比例 2.50:1
- 主色板：#100f16 black 24% / #363432 charcoal 21% / #394c57 charcoal 17% / #716154 brown 16% / #534e48 brown 13%
- 明暗/饱和/对比：brightness 67，saturation 74，contrast 42
- 构图策略：画面信息量较高，适合用中心主体加左右辅助信息，保持缩略图可读。
- 视觉隐喻：把增长机制可视化为飞轮、管道、客户路径、收入仪表盘或团队协作系统。
