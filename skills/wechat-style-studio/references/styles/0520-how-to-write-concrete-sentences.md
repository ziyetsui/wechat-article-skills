## 文章 Skill

### 风格指纹

- 内容类型：步骤型深度指南
- 主题域：综合爆款内容
- 开头机制：用数字、收益或结果开场，把可信度和阅读收益前置。
- 结构骨架：先定义目标和适用人群，再按流程拆步骤，最后给执行清单。
- 段落节奏：中短段为主，解释和行动句交替，适合公众号快读。
- 语气判断：解释型：克制但有判断，先拆问题，再给框架和可执行建议。
- 小标题样本：① 加入专有名词 / ② 把动词拆解到能看见对方身体动作的程度 / ③ 像定义状态一样写 / ④ 加入数字 / ⑤ 提供比较基准 / 不要把自己的麻烦推给读者
- 文本规模：约 525 字符，10 个段落，6 个标题，16 个列表项，0 个引用块
- 爆款承诺：通过遵循这五条简单的商务写作原则，告别模糊不清的邮件与报告，不再让同事感到困惑。

### 可复用文章提示词

```text
请使用「如何撰写职场中的具体化语句风格双技能」的文章 skill 写一篇中文公众号文章。

主题：{填写新主题}
目标读者：{填写目标读者}
核心承诺：{读者读完会获得什么改变或判断}

写作方式：
1. 开头：用数字、收益或结果开场，把可信度和阅读收益前置。
2. 类型：采用「步骤型深度指南」；主题域优先靠近「综合爆款内容」。
3. 结构：先定义目标和适用人群，再按流程拆步骤，最后给执行清单。
4. 节奏：中短段为主，解释和行动句交替，适合公众号快读。
5. 语气：解释型：克制但有判断，先拆问题，再给框架和可执行建议。
6. 小标题参考：① 加入专有名词 / ② 把动词拆解到能看见对方身体动作的程度 / ③ 像定义状态一样写 / ④ 加入数字 / ⑤ 提供比较基准 / 不要把自己的麻烦推给读者
7. 论证时先给判断，再给机制、例子或步骤；不要只堆观点。
8. 每个核心段落都要回答一个隐含问题：读者为什么现在要关心、怎么判断、下一步怎么做。
9. 结尾要把全文收束成一个可执行动作、一个判断标准，或一个值得收藏的清单。

避免：
- 不要写成泛泛鸡汤或百科说明。
- 不要把标题里的承诺摊薄，正文必须持续兑现「通过遵循这五条简单的商务写作原则，告别模糊不清的邮件与报告，不再让同事感到困惑。」。
- 不要堆砌过多并列概念；每一节都要服务于主线。
- 不要使用夸张营销话术替代具体机制。
```
